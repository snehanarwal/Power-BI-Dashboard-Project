# Power BI Dashboard Project

## Project Overview

This project is a Power BI dashboard built to explore and understand the dataset through clear and interactive visualizations. The goal was to turn raw data into simple insights that help in better decision-making.

## Tools Used

* Microsoft Power BI
* DAX
* Excel / CSV dataset

## Features

* Interactive dashboard with filters
* Key performance indicators (KPIs)
* Trend and comparison analysis
* Easy-to-understand visual charts
